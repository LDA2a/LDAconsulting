professions libérales.

## 🧱 Stack

Site 100 % statique, sans dépendance ni build :

- `index.html` — structure et contenus
- `styles.css` — design (palette bleu / blanc / gris) et responsive
- `script.js` — menu mobile, header au scroll, animations d'apparition

Ouvrez simplement `index.html` dans votre navigateur.
